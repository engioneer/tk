보미마미 - 실시간 육아돌보미 중개 및 일정 관리 시스템


0. 개발환경
  • OS/Middleware : Windows XP, Tomcat 6.0
  • Languages/DBMS : Java, JSP, XML, Oracle 10g
  • Libraries/Frameworks : JDK1.6, Android SDK, Spring, MyBatis
  • Tools/Etc : NAVER SVN, eXERD, SQLGate, GCM
  • System Architecture : Model 2 Web Architecture (MVC)

1. 개요
경기 침체와 여성의 사회 참여도 증가로 해마다 맞벌이 부부가 늘고 있는 요즘, 그와 비례하여 돌보미 이용 가구수 역시 지속적으로 증가하고 있는 추세입니다.
하지만 신청 자격 기준이 까다롭고 돌보미와 연결되기까지 대기 시간이 긴 정부 돌보미 사업이나, 임금이 제 각각이고 수수료가 높은 민간 돌보미 업체 등 기존 육아돌보미 서비스의 문제점이 많은 실정입니다.
보미마미는 이에 대한 대체방안으로써 육아 문제라는 현대의 대표적인 사회 문제의 해결에 도움이 되기 위해 개발된 모바일 애플리케이션입니다.

2. 주요 기능
1) 돌보미 검색 및 신청 기능
성별, 나이, 지역 등 돌보미 정보를 조건에 따라 검색하여 서비스 신청

2) 돌보미 연결 기능
부모가 돌보미에게 신청 시 푸시 알림으로 돌보미가 실시간으로 승인 가능

3) 일정 관리 기능
부모가 아이 일정을 등록하면 돌보미가 이를 수행하고 완료 여부 체크 및 코멘트를 남기면 실시간으로 부모에게 푸시 알림

4) 이용후기 기능
사진과 함께 이용후기를 남길 수 있는 게시판 기능

3. 담당 역할
4인으로 구성된 프로젝트 팀의 팀장으로서 수시로 회의를 열어 개발 업무를 분배하고 프로젝트 일정을 관리하였습니다. ERD 툴로 eXERD를 사용하여 DB를 모델링하고 SQLGate로 DB를 관리하였으며, 형상관리 툴인 네이버 SVN을 통해 코드의 변경을 관리하는 등 전반적인 프로젝트 설계 및 관리 업무를 맡아 팀원들을 이끌었습니다. 세부 개발 담당으로는 Spring, MyBatis Framework를 사용하여 서버의 개발을 담당하였으며, 동시에 안드로이드와 DB 파트를 연결하여 개발 업무의 총 책임을 맡았고, 프로젝트 발표자로서 프로젝트를 성공적으로 마무리하였습니다.

원센텐스 - 문장 저장 및 관리, 공유 서비스


0. 개발환경
  • OS/Middleware : Windows 7, Tomcat 6.0
  • Languages/DBMS : Java, JSP, XML, HTML 5, CSS 3, JavaScript, AJAX, MySQL
  • Libraries/Frameworks : JDK1.7, Android SDK, Spring, MyBatis, RESTful API, jQuery
  • Tools/Etc : GitHub, Jenkins, Maven, MySQLWorkbench, AWS
  • System Architecture : Model 2 Web Architecture (MVC)

1. 개요
경제성장 둔화로 자기계발에 대한 관심은 높아지고 있으나 특정 주제의 개인 데이터 보관 및 관리 서비스가 부족한 시장 상황에서, 개인의 자기계발 및 정보수집 니즈에 착안하였습니다.
원센텐스는 책, 영화, 강연 등 매일 접하는 수 많은 매체 정보 속에서 공감이 되고 기억하고 싶은 문장을 쉽고 효율적으로 분류, 저장하고 관리하며 공유하는 자기계발 모바일/웹 애플리케이션 서비스입니다.

2. 주요 기능
1) 체계적인 문장 저장 및 조회 기능
책, 영화, 음악, 강연 등 출처에 따른 카테고리 별 분류 및 개인이 생성한 폴더 별 분류에 따라 문장을 저장하고 조회할 수 있습니다.

2) 출처 검색 기능
포털 사이트 등에서 제공하는 검색API를 사용하여 각 카테고리 별로 책, 영화, 음악, 인물 등 아이템을 검색하여 출처를 관리합니다.

3) 문장 공유 기능
아이템 및 문장의 인기 순위 통계를 알아볼 수 있으며 저장한 문장을 공유하고 타인이 공유한 문장을 공감할 수 있는 SNS를 지원합니다.

3. 담당 역할
개발자 7인, 기획자 1인, 디자이너 1인으로 구성된 스타트업에서 `원센텐스`의 개발 팀장을 맡고 있습니다. RESTful API 설계 및 documentation 작성부터, Spring, MyBatis Framework를 사용한 백엔드 개발 및 JavaScript, JQuery, Ajax를 사용한 프론트엔드 개발을 담당하고 있으며, Maven을 통한 라이브러리 관리, AWS의 RDS 및 MySQLWorkbench를 통한 DB 관리, GitHub을 통한 형상 관리, CI 툴인 Jenkins를 통한 빌드 및 배포 관리 등의 업무를 맡고 있습니다. 스케줄 툴인 Trello를 이용하여 태스크 및 이슈 관리를 하고 있으며, 많은 커뮤니케이션과 정기적인 회의의 진행을 통해 각 파트의 개발 이슈 처리와 업무 분배 및 일정 관리 등 개발 총괄 관리를 담당하고 있습니다.
